# 项目说明文档

## 项目简介
画字小天地是一款面向儿童的象形汉字学习网页应用。提供默认15字与自定义最多10字两种模式，支持本地文件加载并调用火山引擎生成简洁SVG动画。点击图像变字、进度与完成弹窗提示，交互自然、风格简约童趣。

## 概览
- 名称：画字小天地
- 目标：用象形动画帮助儿童直观理解基础汉字
- 形态：单页网页应用，轻量、即用即学

## 核心功能
- 模式切换：默认学习（内置15字）与自定义学习（最多10字）
- 自定义生成：加载本地文件或手动输入，自动为每字生成SVG动画
- 学习交互：点击卡片图像淡出并显字；全部学习完成弹窗提醒
- 进度提示：生成过程顶部轻提示显示进度与错误信息
- 模式指示：右上角常显当前模式（默认/自定义）

## 使用指南
- 打开：`http://localhost:5500/demo.html`
- 导航：点击页面左/右空白切换上一/下一字
- 学习：点击卡片中图像，动画淡出后显示汉字、拼音、释义
- 模式菜单：双击页面空白弹出菜单，选择模式
- 自定义学习：
  - 加载本地文件：在面板点击“加载并生成”，选择 `.txt`/`.json`
  - 手动输入：在文本框输入汉字后点击“生成动画”
- 关闭面板：任意点击面板外区域自动关闭

## 自定义文件格式
- 文本文件：从内容中提取中文字符，取前10个
- JSON 文件：
  - 数组：`["山","日","月",...]`
  - 对象：`{"chars":["山","日","月",...]}`
- 示例：`d:\trae_suzhou\custom_chars.json`

## 接口集成
- 服务：火山引擎 Chat Completions（模型 `doubao-seed-1-6-vision-250815`）
- 用途：根据汉字生成简洁、适合儿童的 200×200 视口 SVG 路径
- 密钥设置：
  - 长按底部“点击图片变变变”约 0.9 秒打开设置框
  - 或按 `Ctrl+Shift+K` 打开设置框
- 存储：`localStorage` 键 `VE_ARK_API_KEY`（仅本地）
- 错误处理：网络或鉴权失败时提示并回退到默认SVG

## 交互与提示
- 顶部轻提示：显示生成进度与错误信息，自动淡出
- 弹窗：生成完成与全部学习完成均弹窗提示，点击可关闭

## 技术栈
- 前端：原生 HTML/CSS/JavaScript
- SVG：原生节点生成与渲染
- 状态管理：本地状态对象与集合记录学习进度
- 存储：`localStorage`（接口密钥）

## 部署与预览
- 本地预览：在项目根目录运行 `python -m http.server 5500`，访问 `http://localhost:5500/demo.html`
- 静态部署：上传页面至任意静态站点即可
